# 汇率 x * y = c

固定乘积   coin x amount   * coin y amount  =  c   c是常量     x y 一个币对  coin a  coin b

usd   rmb 币对     swap

交互前银行的钱 100 usd   720 rmb       100 * 720 = 72000     c 72000 常量    常量是做swap这操作 不会改变这个值
